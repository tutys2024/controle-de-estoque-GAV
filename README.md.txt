# StockControl PWA

Uma Progressive Web App para controle de estoque integrada com Google Sheets.

## Como Hospedar no GitHub Pages

### Passo a Passo

1. **Crie o Repositório**:
   - Acesse o GitHub e clique em "New Repository".
   - Nomeie como `stockcontrol-pwa` e inicialize com um README.
   - Copie o URL do repositório (ex.: `https://github.com/<seu-usuario>/stockcontrol-pwa.git`).

2. **Configure Localmente**:
   - Crie uma pasta no seu computador e abra um terminal nela.
